# Quick Font Reference

## Current Typography

### Body Text: Philosopher
```
Font: Philosopher
Origin: Ancient Greek Agora typeface
Style: Elegant serif
Weights: 400, 700
Usage: All body text, buttons, inputs, labels
```

### Headings: Cinzel Decorative
```
Font: Cinzel Decorative
Origin: Classical Roman inscriptions
Style: Decorative serif
Weights: 400, 700, 900
Usage: All headings (h1-h6)
```
