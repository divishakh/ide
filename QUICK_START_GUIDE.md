# Quick Start Guide - Athena's Code Chambers

## üöÄ Getting Started in 3 Steps

### Step 1: Create a Project
**A project is a container for your code files**

**Method 1 - From Empty State:**
- When you first open the IDE, you'll see a welcome screen
- Click the **"Create Your First Project"** button

**Method 2 - From Sidebar:**
- Look for the **purple "+" button** next to "FILES" in the left sidebar
- Click it to open the project creation dialog

**What to enter:**
- **Project Name**: e.g., "My First App", "JavaScript Practice"
- **Description** (optional): e.g., "Learning JavaScript basics"
- Click **"Create"**

### Step 2: Create a File
**Files contain your actual code**

**Method 1 - From Empty State:**
- After creating a project, you'll see a **"Create New File"** button
- Click it to create your first file

**Method 2 - From Toolbar:**
- Make sure a project is selected (click on it in the sidebar)
- Click the **"New File"** button in the toolbar

**Method 3 - From Sidebar:**
- Click the **"+" button** next to your project name in the sidebar

**What to enter:**
- **Filename**: e.g., "script.js", "app.py", "main.cpp"
- File extension is automatically added if you don't include it
- Click **"Create"**

### Step 3: Write and Run Code
**Now you're ready to code!**

1. **Write your code** in the editor
2. **See suggestions** appear automatically as you type (like VS Code!)
3. **Click "Run Code"** button (green button in toolbar)
4. **View output** in the Console panel on the right

## üêõ Troubleshooting

### "No Project Selected" Error
**Problem:** Clicking "New File" shows this error
**Solution:** 
1. First create a project (click "+" next to FILES)
2. Then create files in that project

### No Projects Showing
**Problem:** Sidebar is empty
**Solution:**
1. Click "+" button next to "FILES"
2. Create your first project
3. Or click "Create Your First Project" in center

### Autocomplete Not Working
**Problem:** No suggestions appearing
**Solution:**
1. Just keep typing - suggestions appear instantly
2. Press Ctrl+Space to trigger manually
3. Make sure you're typing valid code
4. Try typing common keywords: "function", "const", "let"

### Code Won't Run
**Problem:** Nothing happens when clicking "Run Code"
**Solution:**
1. Check for syntax errors (red underlines)
2. Make sure file is selected
3. Check console for error messages
4. Try simple code first: `console.log("test")`

### Dark Mode Too Dark
**Problem:** Dark mode is too black
**Solution:**
- The dark mode now uses soft blue-gray colors!
- Background: Soft dark blue-gray (#1a1f2e)
- Much easier on the eyes
- Toggle with moon/sun icon to see the difference
